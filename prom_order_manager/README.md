# Интеграция Prom.ua и Telegram для обработки заказов

Этот модуль автоматически обрабатывает заказы с Prom.ua, отправляя уведомления закупщику в Telegram.

## Функционал

1. **Авто-принятие заказов**: Новые заказы (статус "Новый") автоматически переводятся в статус "Принят" (received).
2. **Мониторинг**: Скрипт проверяет заказы со статусом "Принят".
3. **Уведомление**: Если у заказа появился ТТН (декларация), скрипт формирует сообщение и отправляет его в Telegram.

## Формат сообщения

```text
Поставщик (из личных заметок)
Размер Цвет (из заказа)
Артикул/Модель (из заметок или товара)
Цена закупки (из заметок)
ТТН + Имя Клиента
```

## Установка и Настройка

1. **Папка**: `prom_order_manager` (создана на рабочем столе).
2. **Настройка**:
   - Откройте файл `.env` в этой папке.
   - Вставьте ваш токен Prom.ua: `PROM_API_TOKEN=ваш_токен`
   - Вставьте токен Telegram бота: `TELEGRAM_BOT_TOKEN=ваш_бот_токен`
   - Вставьте ID чата закупщика: `TELEGRAM_CHAT_ID=id_чата`
3. **Запуск**:
   - Установите библиотеки (если не установлены): `pip install -r requirements.txt`
   - Запустите скрипт: `python main.py`

Скрипт будет работать в фоне, проверяя заказы каждые 5 минут.
